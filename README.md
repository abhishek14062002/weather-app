# weather-app
Using HTML5,CSS3,JS
Features:
Developed a Weather Application using HTML, CSS, and JavaScript with a clean and responsive user interface.

Integrated real-time weather data API to display temperature, humidity, wind speed, and weather conditions.

Implemented city-based search functionality with dynamic data rendering.

Designed a mobile-friendly layout using CSS for cross-device compatibility.

Handled API errors and invalid inputs to improve user experience.

Optimized JavaScript logic for faster data fetching and smooth performance
